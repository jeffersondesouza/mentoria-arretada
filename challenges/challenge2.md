# Challange 2

No Segundo vamos inciar nosso projeto desafio. Não vamaos nos preocupar com estilização nesse momento, vamos focar só na estrutura do html.
## Foco:
- HTML

## Tarefas:
 - Crie um novo reposiório no seu github com o nome "encurtador-de-links";
 - Clonar o repositório localmente;
 - Localmente, criar um arquivo chamado index.html;
 - Criar a estrutura básica (h1, p, form) do nosso encurtador. Se basear na [Imagem](https://raw.githubusercontent.com/jeffersondesouza/frontend-challenges-handson/master/chaordic/html-css-js-version/assets/functional.gif) 
 - fazer commit das mudanças e subir essas mudanças;
 
    ```git add index.html``` 

    ```git commit -m"Estrutura inicial do encurtador"``` 

    ```git push origin main```
