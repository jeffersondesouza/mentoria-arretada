DESCRIÇÃO:
- Faremos um Buscador de CEP;
- O usuário irá digitar o CEP e clicar no botão preto;
- A aplicação deverá consultar a API [viacep](https://viacep.com.br/), basicamente basta fazer a requisição para a [API do viacep]( https://viacep.com.br/ws/CEP-QUE_DESEJA/json/) e ele retornará um JSON com o CEP
Uma vez que tem o CEP basta mostrar as informações
ATENÇÃO: o map no que você vê deve ser apenas uma imagem, ela está na pasta de assets, num futuro, faremos uam evolução desse desafio integrando ao google maps.
